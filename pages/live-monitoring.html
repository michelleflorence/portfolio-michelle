Automa Live Monitoring Dashboard: Simplifying Real-Time Fleet Tracking

The Automa Live Monitoring Dashboard makes it easy to track and monitor fleets in real-time. With live updates, you can stay on top of fleet movements, and the playback feature lets you revisit delivery routes through interactive video simulations. You can also filter and view statuses based on specific fleets or orders for a more focused experience. Built using React JS, this project combines functionality with a clean, user-friendly interface designed from Figma prototypes, ensuring a seamless experience.
