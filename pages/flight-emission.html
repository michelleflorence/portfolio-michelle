Injourney Flight Emission: Building a Greener Future One Flight at a Time

Injourney Flight Emission is a project developed for a company called Injourney, aimed at calculating flight emissions. The process started with designing the interface in Figma, which was then converted into functional components using React JS. Right now, the project is still in the mockup phase and hasn’t been integrated with the backend. It’s focused on showcasing the design and front-end functionality as a foundation for further development.
